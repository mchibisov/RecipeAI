# RecipeAI — Информационная система поиска рецептов с интеграцией LLM

Веб-приложение для авторизованного поиска кулинарных рецептов по базе данных с возможностью генерации интерактивных ответов, кулинарных советов и модификации инструкций искусственным интеллектом в режиме реального времени.

## Технологический стек
* **Backend:** Python 3.11+, Flask (WSGI-фреймворк), SQLAlchemy (ORM)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Asynchronous Fetch API, DOM API)
* **Database:** PostgreSQL (хранение учетных записей пользователей и активных сессий)
* **AI Integration:** Mistral AI API (модель для генерации ответов по контексту рецепта)

## Архитектура и безопасность
* **Аутентификация:** Сессионная на базе токенов. Токены генерируются на стороне бэкенда и сохраняются в клиентских куках с флагами безопасности `HttpOnly` (защита от XSS) и `SameSite=Lax` (базовая защита от CSRF).
* **Безопасность данных:** Пароли пользователей хэшируются перед сохранением в базу данных (clear-text пароли не хранятся).
* **Frontend Security:** Реализовано клиентское экранирование HTML-сущностей (`escapeHtml`) для предотвращения уязвимостей типа Stored/Reflected XSS при обработке данных из внешних API.

## Локальное развертывание
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt

   ```

2. Настройте конфигурационные переменные в файле `.env` (API-ключи, строка подключения к БД).
3. Запустите приложение:
```bash
python app.py

```



## Развертывание в облаке (Deployment)

Приложение подготовлено для деплоя на платформы облачного хостинга (Render, Railway, Koyeb).

* Точка входа для запуска WSGI-сервера адаптирована под динамическое чтение переменной окружения `PORT`.
* Сетевой интерфейс переведен на хост `0.0.0.0` для приема внешних HTTP-запросов из глобальной сети.
